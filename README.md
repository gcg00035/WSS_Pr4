# Población Automática de Ontología Corporativa

## Replicación del Entorno (Fijación de dependencias)

Para garantizar la replicabilidad exacta del proyecto, se ha utilizado un entorno virtual (`virtualenv`) y se han fijado las versiones en el archivo `requirements.txt`.

### Pasos para ejecutar:
1. Crear y activar el entorno virtual
2. ejecutar `pip install -r rrequirements.txt`
3. Ejecutar el script `python poblar_ontologia.py`
